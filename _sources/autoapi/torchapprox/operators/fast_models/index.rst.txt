.. py:function:: mul8s_1kv8(base_func, op1, op2, kwargs)

   Approximate Multiplication accurate fast model.
   This yields the same output as the baseline layer.
   Primarily useful for testing

   :param base_func: Basic layer function
   :param op1: 1st operand
   :param op2: 2nd operand
   :param kwargs: Keyword arguments passed to basic layer function
                  (i.e. configuration parameters for Conv2d)

   :returns: Accurate Layer operation result
